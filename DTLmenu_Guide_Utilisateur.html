DTLmenu — Guide utilisateur

Guide du lanceur graphique de compilation de la suite NetDTL.

Windows HTAPyInstallerSuite NetDTL

Objectif

DTLmenu permet de compiler les outils Python de la suite NetDTL depuis une fenêtre unique, sans ressaisir les commandes PyInstaller à la main.

Préparer l’environnement

  1. Vérifier que Python est installé.
  2. Vérifier que PyInstaller est disponible.
  3. Vérifier que les projets de la suite sont présents dans le même dossier racine.

Définir le dossier racine

Le champ `ROOT_DIR` doit pointer vers le dossier qui contient les projets de la suite. Une fois validé, DTLmenu mémorise ce chemin dans `DTLmenu.root`.

Si un projet n’est pas trouvé, commencer par vérifier le dossier racine plutôt que les commandes de build.

Choisir un build

Build individuel

Chaque outil Python peut être compilé séparément : DTLknowsWhy, DTLsaysWhat, GitDTL, GitHubMenu ou DTLaudit.

Build complet

Le build complet enchaîne les compilations principales de la suite.

NetDTL

NetDTL est un projet PHP/MySQL : il n’a pas besoin de compilation PyInstaller.

Où trouver les résultats

Les exécutables générés sont placés dans les dossiers `dist` des projets concernés. DTLmenu propose des accès rapides vers ces dossiers.

Dépannage courant

  • Si PyInstaller manque, installer ou mettre à jour PyInstaller.
  • Si un build échoue, vérifier les dépendances Python du projet concerné.
  • Si le dossier `dist` n’apparaît pas, vérifier que la compilation est terminée sans erreur.
  • Si DTLmenu ne trouve pas les projets, corriger `ROOT_DIR`.
